Informações importantes:

O site é excelente para descobrir e entender as tags.

Na maioria das tags, é necessário possuir uma tag de abertura <...> e uma tag de fechamento </...>. É extremamente importante não esquecer de fechar as tags para não bagunçar os códigos.

Tipos de linguagem de programação

HTML

É como a espinha dorsal do corpo humano aplicada à programação

Tags importantes:

  • <Head>

    Indica a "cabeça" do produto, onde será identificado o nome do produto e a linguagem utilizada.

  • <Title>

    É preciso "abrir e fechar" para indicar, no meio, qual será o título do site.
    É usado dentro do head.

  • <Body>

    Tudo que vai estar dentro do site. Todas as informações precisam estar dentro dessa chave para aparecerem no site.

  • <H1> a <H6>

    Define o tipo de título usado, sendo o H1 o maior e de maior destaque, e o H6 o menor.

  • <P>

    Funciona como um parágrafo dentro do texto colocado

  • <IMG>

    Para acrescentar uma imagem na sua página, sempre acompanhado por src = "nome do arquivo salvo dentro da pasta" (source, ou fonte em inglês). Pode possuir também alt (para apresentar texto alternativo), width (largura em pixels) e height (altura em pixels) ainda dentro da tag da imagem. Ela não precisa de uma tag de fechamento.

CSS

É a perfumaria da programação.
Traz cores, estilos e demais informações para embelezar e tornar o produto mais atrativo.

Um site muito interessante para visualizar a aplicação do CSS é o CSS Zen Garden.

O CSS é um segundo passo após o HTML, logo ele será usado dentro do head.

ID é um seletor para colocar uma propriedade específica em uma única parte da página. Deve ser inserido dentro da tag no body.

SPAN é um seletor usado para colocar uma propriedade específica em mais de uma parte da página, Deve ser inserido dentro das tags no body.

JavaScript

Conectividade e integrações.
Funciona como a "alma"do produto, fazendo as ligações e funções necessárias.
